Memegenerator app

HTML & CSS

GitHub Repo Live version

This is the interactive Meme generator that fetches the images from API and allows user to enter the text that shows up on top and text that shows up on bottom of the image, when user clicks on get meme image button, it will render the image with text on top and bottom. It is built using ReactJs, vanilla CSS, and is utilizing Flexbox and CSS Variables. It was built as a part of Scrimba's Frontend Developer Career Path.
